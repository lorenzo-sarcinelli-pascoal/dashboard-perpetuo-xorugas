# Icones do atalho (THE Brand System)

Os icones desta pasta seguem a marca **Tracking HighEnd**:

- **Fonte do vetor:** mesmo path do **Ícone** (bege sobre escuro) do guia [branding.trackinghighend.com](https://branding.trackinghighend.com/) — seção Identidade / Logo / Ícone.
- **Cores:** MAJOR01 `#D9C8B7` sobre MAJOR02 `#0F172A` (paleta oficial).

Arquivos:

| Arquivo | Uso |
|--------|-----|
| `the-icon.svg` | Favicon vetorial (navegadores modernos) |
| `app-icon.png` | 512×512 — manifest PWA / Android |
| `apple-touch-icon.png` | 180×180 — atalho na Home do iPhone |

Para atualizar quando o guia de marca publicar nova versao do icone: baixe o SVG oficial na pagina de branding e substitua `the-icon.svg`, depois regenere os PNGs (512 e 180) ou copie os exports oficiais mantendo os nomes acima.
